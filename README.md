# ClothShare

A clothing donation platform that connects donors with donation centers using AI-powered matching.

## Getting Started

To run this project locally:

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd ClothShare

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase
- AI-powered matching algorithm

## Features

- User authentication
- Clothing donation form
- AI-powered center matching
- Location-based recommendations
- Real-time donation tracking
